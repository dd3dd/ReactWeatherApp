# 天氣預報App

## --使用前需打開瀏覽器定位功能--

模仿iphone的天氣App介面，提供現在位置的天氣、24小時內每小時天氣預報以及10天天氣預報。

## --因WeatherAPI trial plan到期，免費版本只提供未來3天預測--
